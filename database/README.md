# MedIntel Database

## Overview

This folder contains the MySQL database scripts used by the MedIntel application.

## Database

- Database: MySQL 8.0
- Name: medintel

## Tables

- medicines
- medicine_brands
- medicine_dosage
- medicine_indications
- medicine_interactions
- medicine_precautions
- medicine_safety
- medicine_side_effects
- medicine_storage

## Usage

Import the SQL files into a MySQL database named **medintel** before running the backend application.
