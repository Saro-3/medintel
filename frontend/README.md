# MedIntel Frontend

## Overview

This folder contains the frontend application of **MedIntel**, developed using **React.js** and **CSS**. It provides a responsive user interface for searching medicines, viewing medicine details, managing inventory, and tracking orders through REST API communication with the Spring Boot backend.

## Technologies Used

* React.js
* CSS3
* React Router
* Axios
* JavaScript (ES6)

## Folder Structure

```text
src/
├── assets/
├── components/
├── pages/
├── services/
├── context/
├── hooks/
├── utils/
├── App.jsx
└── main.jsx
```

## Installation

1. Navigate to the frontend folder.

```bash
cd frontend
```

2. Install dependencies.

```bash
npm install
```

3. Start the development server.

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## Features

* Medicine Search
* Medicine Information Display
* Inventory View
* Order Tracking
* Responsive User Interface

## Backend Connection

The frontend communicates with the Spring Boot backend through REST APIs using Axios.
